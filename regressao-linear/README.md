# 🧠 Aprendizado de Máquina - Regressão Linear

Este repositório contém implementações práticas dos conceitos estudados na disciplina de **Aprendizado de Máquina**, com foco em problemas de **regressão**.

---

## 📌 Objetivo

Aplicar técnicas de Machine Learning para prever valores contínuos, explorando desde modelos simples até abordagens mais robustas.

---

## 📂 Projetos

### 🏠 Regressão Linear Simples - Preço de Casas

Modelo básico para prever o preço de casas com base na área em metros quadrados.

* 📥 Entrada: Área (m²)
* 📤 Saída: Preço (R$)
* 📈 Modelo: Regressão Linear
* 📊 Visualizações:

  * Dispersão dos dados
  * Reta de regressão
  * Comparação entre valores reais e preditos

---

### 🏢 Desafio do Aluguel - Regressão Múltipla

Problema mais realista envolvendo múltiplas variáveis para prever o valor de aluguel de imóveis.

* 📥 Entradas:

  * Tipo do imóvel
  * Bairro
  * Quartos
  * Vagas
  * Área
  * Condomínio
  * IPTU
* 📤 Saída: Valor do aluguel

---

## 🤖 Modelos Utilizados

* 🔹 Regressão Linear
* 🌲 Random Forest Regressor

---

## 📊 Avaliação

Os modelos foram avaliados utilizando:

* 📉 MSE (Erro Quadrático Médio)
* 📊 R² (Coeficiente de Determinação)

---

## 📈 Principais Análises

* Comparação entre modelos (Linear vs Random Forest)
* Visualização de dados (scatter plots e histogramas)
* Importância das variáveis (feature importance)

---

* A regressão linear apresentou bom desempenho em cenários simples
